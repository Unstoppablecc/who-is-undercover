# 谁是卧底联机版使用说明

## 1. 配置 DeepSeek Key

复制 `.env.example` 为 `.env`，填写：

```env
DEEPSEEK_API_KEY=你的 DeepSeek API Key
HOST=0.0.0.0
PORT=8000
```

## 2. 启动服务

```powershell
python server.py
```

电脑本机访问：

```text
http://127.0.0.1:8000
```

手机同 Wi-Fi 访问：

```text
http://电脑局域网IP:8000
```

公网访问：

```text
http://服务器公网IP:8000
```

**注意**：如果无法访问，请检查防火墙设置：

```bash
# Ubuntu/Debian 开放端口
sudo ufw allow 8000

# 或使用 iptables
sudo iptables -A INPUT -p tcp --dport 8000 -j ACCEPT
```

## 3. 游戏功能

### 创建房间
- 输入昵称（最多5个字）
- 点击"创建房间"按钮
- 房主可设置：玩家人数、卧底数量、空白词数量、词语类型、难度等

### 加入房间
- 输入房间号和昵称
- 点击"加入房间"

### 房主设置
- **玩家人数**：3-15人
- **卧底数量**：1至玩家人数-1
- **空白词数量**：0至玩家人数-卧底数量-1
- **词语类型**：零食饮料、外卖美食、奶茶咖啡等（支持自定义）
- **难度**：低难度、中等难度、高难度、超高难度
- **模型**：Flash（速度快）、Pro（质量高）

### 游戏流程
1. 所有玩家准备后，房主点击"开始游戏"
2. 每位玩家会看到自己的词语（卧底看到不同词语）
3. 轮流描述自己的词语，但不能说出词语本身
4. 点击其他玩家的投票按钮进行投票
5. 所有存活玩家投票结束后，得票最多的玩家出局
6. **平票规则**：平票且≥2票时，平票玩家同时出局
7. **自投规则**：玩家可以投票给自己
8. 存活≤3人时，已出局的玩家可以参与投票

### 爆词功能
- 猜到平民词时，可输入词语并点击"校验"
- 爆词成功/失败结果会广播给所有玩家

### 房间管理
- 点击"退出房间"可离开房间
- 房主离开后，房间自动解散
- 点击"重开"可重新开始游戏（房主权限）
- 房主可在游戏开始后修改部分设置

## 4. 界面说明

### 玩家卡牌
- 显示玩家编号、昵称
- 出局玩家显示身份（平民出局/卧底出局/空白出局）

### 投票面板
- 显示投票进度（已投票/存活人数）
- 投票结束后显示：
  - 出局结果
  - 得票统计（使用昵称）
  - 投票详情（A→B 表示 A 投给了 B）

### 我的卡牌
- 点击查看词语（再次点击隐藏）
- 颜色变化表示查看次数

## 5. 常见问题

### 手机无法加入房间
检查 `.env` 配置：

```env
HOST=0.0.0.0
```

`127.0.0.1` 只允许本机访问，手机无法连接。

### 刷新页面
刷新浏览器会使用本地保存的 `clientId` 自动恢复原玩家，不会新增玩家。

### 人数不足无法开始
开始游戏时，后端会校验实际加入人数是否等于设置人数，不一致时无法发牌。

## 6. 部署到服务器

### Ubuntu 开机自启动（systemd）

1. 创建服务文件：
```bash
sudo nano /etc/systemd/system/wodi.service
```

2. 写入内容：
```ini
[Unit]
Description=Who is the Undercover Game Server
After=network.target

[Service]
Type=simple
User=你的用户名
WorkingDirectory=/path/to/wodi
ExecStart=/usr/bin/python3 /path/to/wodi/server.py
Restart=on-failure
RestartSec=5
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

3. 启用服务：
```bash
sudo systemctl daemon-reload
sudo systemctl enable wodi
sudo systemctl start wodi
```

### Nginx HTTPS 配置（可选）
公网建议使用 Nginx 反向代理并配置 HTTPS。
